# Portfolio | Valentin Fernandez

Portfolio personal desarrollado con Next.js y Tailwind CSS para presentar proyectos full stack, stack técnico y vías de contacto profesional.

## Stack

- Next.js
- React
- Tailwind CSS
- React Icons

## Secciones

- Hero con propuesta de valor y enlaces de contacto
- Proyectos destacados como mini casos de estudio
- Stack técnico
- Sobre mí
- Contacto

## Ejecutar en local

```bash
npm install
npm run dev
```

Abrir `http://localhost:3000`.

## Build

```bash
npm run build
```
